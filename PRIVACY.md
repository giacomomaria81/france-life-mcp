# Privacy Policy — France Life MCP

**Last updated:** April 17, 2026

## What this service does

France Life MCP is an open-source tool that queries free, public French government APIs on behalf of the user. It acts as a passthrough — it sends your query to the relevant government API and returns the result.

## Data we collect

**None.** We do not collect, store, log, or retain any personal data. No analytics, no tracking, no cookies, no user accounts.

## Data flow

1. You send a query (e.g., "find a pharmacy in Paris")
2. Our server forwards it to the relevant public API (e.g., Nominatim/OpenStreetMap)
3. The API response is returned to you
4. Nothing is stored. No logs are kept.

## Third-party APIs

This service queries the following public, free APIs on your behalf:
- Base Adresse Nationale (api-adresse.data.gouv.fr)
- Open-Meteo (api.open-meteo.com)
- Hub'Eau (hubeau.eaufrance.fr)
- Calendrier gouv.fr (calendrier.api.gouv.fr)
- Éducation Nationale (data.education.gouv.fr)
- OpenDataSoft / Annuaire Santé (public.opendatasoft.com)
- Nominatim / OpenStreetMap (nominatim.openstreetmap.org)
- SNCF Open Data (ressources.data.sncf.com)
- Service-Public.fr (api-lannuaire.service-public.fr)
- Géorisques (georisques.gouv.fr)
- Recherche Entreprises (recherche-entreprises.api.gouv.fr)

Each of these services has its own privacy policy. We encourage you to review them.

## GDPR

Since we do not collect or process any personal data, GDPR obligations are minimal. No data subject requests are applicable as no personal data is held.
